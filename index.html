Debian packaging for BBBS (BBBSDEB)

This project is about creating debian packaging for BBBS, which is a complete BBS package with WWW/InterNet/modem/ISDN/LAN based access modes; see the BBBS home page, http://www.bbbs.net/, for more details.

The installer package bbbs-installer downloads the archive from the www.bbbs.net web site, or uses the one in the tmp directory if already present, then installs the application to the system.

The package bbbs-equivs is for when you have manually installed BBBS and would like to make it like a standard Debian installed package.

There is also the BTERM program from the same authors; packaging for that has been started but is not very far along yet.
