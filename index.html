Rocasa BBS

The Rocasa BBS is a system accessible as both a traditional Bulletin Board System, via telnet, as well as via the Web.

Rocasa BBS Index

Basic user accounts here are free, but not anonymous... To establish an account; you simply use a telnet-ssl capable client to telnet to the BBS at bbs.rocasa.us on the standard telnet-ssl port 992 and log in as a new user, or you an use a web browser to register as a new user. (Note that I require real first and last names, & other information, for you to keep the account open...) Once your account is active, you will have several ways of accessing the bbs via the internet:

    To log in, you can use:
  • You can go to the HTTPS Login page, using the standard port 443, for secure access via a web browser like Firefox or Internet Explorer.
  • Telnet-ssl, to bbs.rocasa.us using port 992.
  • Guest access via the web: Read Only access to files & some conferences
    • HTTP or HTTPS: Guest login
    • Once there; click on the Join/Resign Message Areas link to display the full list of conferences available.
  • For more about Rocasa BBS, http://bbs.rocasa.us/about.html
  • For more about a program usable for telnet/ssh on various platforms, http://bbs.rocasa.us/bterminfo.html
  • For information about BBS Software, http://bbs.rocasa.us/software.html
  • For more about the BBBS File Distribution Network, http://bbs.rocasa.us/bbbsfdn.html
  • For more about the Perl File Distribution Network, http://bbs.rocasa.us/perlfdn.html
