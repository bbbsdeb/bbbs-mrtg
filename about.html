About Rocasa BBS

The Rocasa BBS is a system accessible as both a traditional Bulletin Board System, via landline or telnet, as well as via the Web.

It currently has around 400 message areas, with the following areas of interest:
  • International, etc, general chat areas, including multi-lingual
  • Science Fiction
  • Space
  • Ham Radio
  • The Internet, including specific to HTML
  • Hardware, including modems
  • Operating Systems; including Dos, Windows, & Linux
  • Programming; including C/C++, Pascal, & Basic
  • Fidonet related areas
  • Sysops' Tech Net related echos

Basic User accounts here are free, but not anonymous... To establish an account; you simply telnet to the BBS and log in as a new user, or you an use a web browser to register as a new user. (Note that I require real first and last names, & other information, for you to keep the account open...) You can also access the bbs via landline at 1-586-791-1973. Once your account is active, you will have several ways of accessing the bbs via the internet:

  • You can go to the login page, for access via a web browser like Netscape.
  • https://rocasa.us/bbbs , using the standard port 443, for secure access via a web browser like Netscape.
  • Telnet-SSL to bbs.rocasa.us using the standard telnet SSL port of 992.
    • Go here for information about a telnet client that can encrypt connections to this BBS.
  • Guest access: Read Only access to files & some conferences
    • HTTP: Guest (Read Only)
  • For FTN Mailer access to Rocasa BBS:
    • BinkP: rocasa.us, on the non-standard port 24556
    • phone: n/a, at this time
    • If you are looking for or are otherwise interested in an FTNfeed, as a node or a point, please contact me.
  • For statistics of Rocasa BBS, https://rocasa.us/stats/index.html
  • For more about a program usable for telnet/ssh on various platforms, https://rocasa.us/bterminfo.html
  • For more about one experience with BBBS on Linux, https://rocasa.us/linuxbbbs.html
  • For more about the BBBS File Distribution Network, https://rocasa.us/bbbsfdn.html
